# 🎨 阿凱（KAI）角色設定書

> **用途**：給 HeyGen 挑選 / 自訂 Avatar 的依據；也可用於 Midjourney / DALL-E 產出視覺參考圖。  
> **調性**：學長姐親和帶人、有耐心、電銷團隊的自己人

---

## 👤 角色基本設定

| 項目 | 設定 |
|------|------|
| **名字** | 阿凱（KAI） |
| **性別** | 女 |
| **年齡** | 28-32 歲 |
| **身份** | 資深 HRBP / 電銷學長姐 |
| **性格** | 親和、有耐心、溫暖但專業、帶點聰明感 |
| **說話風格** | 像學姊跟學妹聊天，不端架子，但講到重點時很穩 |

---

## 👔 外型設定

| 項目 | 設定 | 備註 |
|------|------|------|
| **髮型** | 中長髮（及肩到鎖骨），自然黑或深棕 | 不要燙太捲，簡單乾淨 |
| **臉型** | 東亞臉孔，圓潤偏溫和，不要太銳利 | 親和感關鍵 |
| **妝容** | 日常職場妝，不濃 | 口紅偏裸色或粉色 |
| **上衣** | 白色或淺藍襯衫 + 針織外套（米色/灰色） | 辦公室休閒感 |
| **配件** | 可戴簡單項鍊，不誇張 | 避免太多裝飾分散注意力 |
| **背景** | 淺色辦公室模糊背景 / 米白純色 / 淺灰漸層 | 不要搶視覺 |

---

## 🎯 為什麼選這個設定？

你選了「學長姐親和帶人、有耐心」的調性，反向推導：

| 元素 | 為什麼這樣選 |
|------|------------|
| **女性 28-32 歲** | 職場上「學姊」定位的甜蜜點，太年輕顯嫩、太年長變主管感 |
| **中長髮** | 比短髮柔和，比長髮專業，「大姊姊」視覺記憶點 |
| **襯衫+針織衫** | 不是正裝（不硬），不是 T-shirt（不隨便），剛好是「上班但放鬆」 |
| **溫和臉型** | 求職者、新人看了不會怕，「我願意問她問題」的感覺 |
| **淺色背景** | 跟圖卡的深藍#1E3A5F形成對比，阿凱出現時視覺聚焦 |

## 🔍 HeyGen Avatar 挑選指南

**進入 HeyGen → Avatars → 篩選條件：**

```
Gender: Female
Ethnicity: Asian
Style: Business Casual / Professional
Age Group: Young Adult / Adult
Action: Presenting / Talking
```

### 我推薦的 Avatar 試用順序（依符合度）

1. **先試 Asian Female + Business Casual + 面帶微笑** 這類關鍵字的 Avatar
2. 搜尋名字含 `Yuki`、`Mei`、`Jenna`、`Amy`、`Sarah-Asian`、`Lisa` 的
3. 避開過於正裝（黑色西裝套裝）或太休閒（T-shirt）的選項
4. **試看 5 個 → 拿同一段 Segment 1 的文字 generate → 比較哪個最順眼**

### Custom Avatar（自訂）？

如果你想要**超級像的 Avatar**，Creator 方案有 **1 個 Instant Avatar 額度**：
- 拍 2-5 分鐘講話影片（找符合氣質的員工或朋友）
- 上傳到 HeyGen
- 幾小時內生出你的專屬阿凱

**但如果你還沒真人模特，先用內建 Avatar 就好，以後再升級。**

如果你想**先產一張視覺參考圖**，看實體長相對不對，再找 HeyGen Avatar，用這個 prompt：

### 中文說明版（貼進中文 AI 繪圖）

```
30 歲東亞女性的肖像，友善溫暖的表情，微笑，
中長直髮（深棕色），白色襯衫搭配米色針織外套，
```

### 多個變化版本（試不同氣質）

**版本 A：更親和**
```

looking directly at camera with kind eyes...
```

**版本 B：更專業**
```
...confident yet gentle expression, slight smile, 
professional demeanor, poised posture...
```

**版本 C：更像學姊**
```
...casual but put-together, like a senior colleague 
who just looked up from her desk, inviting atmosphere...
```
