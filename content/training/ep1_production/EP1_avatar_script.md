## 📝 標記符號說明

文字中包含 HeyGen **特殊控制符號**，有效提升自然度：

- `[pause-short]` → HeyGen 停頓 0.3 秒（段落內換氣）
- `[pause-long]` → HeyGen 停頓 0.8 秒（轉場用）
- `[emph]...[/emph]` → 重音強調
- **半形句號 .** → HeyGen 斷句更自然
- **全形句號 。** → 段落結束

> ⚠️ HeyGen 目前不完全支援 SSML 標籤，上面符號請**手動用 HeyGen 介面的「pause」「emphasis」功能標記對應位置**，或直接複製文字後用語音編輯器微調。

# 🛠️ HeyGen 操作 Cheat Sheet

## 第一次使用（30 分鐘完成第一段測試）

1. **註冊** → [heygen.com](https://heygen.com) 用 Google 帳號登入最快
2. **選方案** → Creator $29/月（先月繳不要年繳）
3. **選 Avatar** → Browse Avatars → 篩 Asian Female → 找一個最接近「親和學姊」氣質的
4. **創建影片** → Video → Blank Video → 貼第一段文字
5. **選聲音** → Voice → Chinese → 試聽幾個女聲挑一個
6. **Preview** → 聽效果，不滿意換聲音再試
7. **Generate** → 產出 → 下載 MP4

💡 前 3 次 generate 會比較卡，之後就順了。

## 常見問題

**Q：中文字慢慢念怎麼調？**  
A：Voice 設定→ Speed → 拉到 -5% 到 -10%

**Q：阿拉伯數字「3」要念「三」還是「three」？**  
A：HeyGen 中文語音預設念中文讀音，如果被念英文可以把「3 萬」改成「三萬」

**Q：credits 快用完怎麼辦？**  
A：Creator 200 credits/月 = 大約 200 秒 premium avatar，**一般 avatar 不扣 credits**，所以整集 18 分鐘用一般 Avatar 完全沒問題
