# 需求探索

Description: 以黃金圈理論慢慢開始挖掘客戶，一切以【瞭解】客戶為主
網址: https://docs.google.com/spreadsheets/d/11yrugPYhcVasu7cZWe7yuNnm8C2z_s8yW93FqDlyQg0/edit?gid=1781133094#gid=1781133094

| **需求探索/痛點 階段** | **行為/心理狀態演變** |
| --- | --- |
| **目標** | 瞭解客戶目前的投資狀況、需求與痛點。 |
|  | 引導客戶認識現有方法的不足，激發參與課程的興趣。 |
| **應注意** | 避免過太多需求痛點去挖掘，讓客戶產生不信任感 |
|  | 請用客觀常理的比喻、舉例方式來進行跟客戶說明 |
| **為什麼這樣做** | 拆解客戶自己本身對於投資上的理解程度，確認他對於投資現狀的領域狀況 |
|  | 增加我們的專業度，面對不同問題出現或者客戶也不清楚的問題可以以客觀常理的方式來去跟客戶舉例及解釋 |
| **話術/實例模擬-基本面** | **詢問現狀** |
|  | 請問您選股時主要依據什麼？例如財報數據、朋友推薦，還是市場熱門股？ |
|  | **挖掘痛點** |
|  | 有沒有遇到過買進後股價下跌，但不知道原因的情況？ |
|  | **提供解決方法** |
|  | 其實很多時候是因為基本面的問題。例如營收連續下滑會提前反映在股價上。我們課程會教您如何快速篩選基本面良好的公司。 |
|  | **實例舉例** |
|  | 比如說，您知道像台積電這樣的公司，除了毛利率穩定超過50%，它的營收年增率也是持續增長的。這些數據幫助投資人對未來的走勢更有信心。 |
|  | **應用情境與客戶反應** |
|  | 客戶可能回應：「但我覺得要分析財報很難，數據看不懂。」 |
|  | 回應話術：「其實只需要關注幾個關鍵數據，例如毛利率和ROE。我們的課程會用案例一步步帶您掌握，學起來非常簡單。」 |
| **話術/實例模擬-技術面需求探索** | **詢問現狀** |
|  | 操作時您會看K線或均線嗎？例如黃金交叉、支撐壓力位？ |
|  | **挖掘痛點** |
|  | 有沒有遇到過買高後被套，或賣出後股價大漲的情況？ |
|  | 提供解決方法 |
|  | 我們的課程會教您如何用技術指標，例如RSI來判斷買賣時機，避免這些問題。 |
|  | 實例舉例 |
|  | 舉個例子，當股價跌破50日均線時通常是賣出的訊號，但如果K線形態顯示支撐位強勁，則可能是反彈的好機會。 |
|  | **應用情境與客戶反應** |
|  | 客戶可能回應：「我之前試過看K線，但還是不知道該什麼時候買賣。」 |
|  | 回應話術：「這就是為什麼我們課程會從基礎開始，例如如何判讀黃金交叉訊號，幫助您一步步建立技術面的分析能力。」 |
| **話術/實例模擬-籌碼面需求探索** | **詢問現狀** |
|  | 你會關注外資或法人的買賣動向嗎？知道這對股價的影響有多大嗎？ |
|  | **挖掘痛點** |
|  | 很多人追高是因為忽略了籌碼分布，例如主力出貨時股價卻短期上漲。 |
|  | **提供解決方法** |
|  | 我們會教您如何判斷主力進出場，找到真正值得投資的標的。 |
|  | **實例舉例** |
|  | 比如說，當外資連續5日買超同一支股票，而融資餘額大幅下降時，通常表示該股後續潛力更高。 |
|  | **應用情境與客戶反應** |
|  | 客戶可能回應：「外資買賣訊息很多，我不知道要看哪些才重要。」 |
|  | 回應話術：「我們的課程會教您如何用簡單的工具，例如追蹤外資買賣超數據，快速找到重點股票。」 |
