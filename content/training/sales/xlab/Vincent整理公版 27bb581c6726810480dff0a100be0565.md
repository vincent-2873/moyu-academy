- 主要聽取內容重點
    1. **快速識別受眾：** 如何透過幾個問題，精準判斷學生的 **需求、痛點與輪廓**。
    2. **建立信任與顧問角色：** 學習如何從「**推銷員**」轉變為「**職涯顧問**」。
    3. **價值塑造與痛點放大：** 了解如何讓學生自己意識到「**學用落差**」與「**作品集的重要性**」。
    4. **完整銷售流程：** 掌握從破冰、發掘需求，到提出方案、處理反對意見的完整架構。
    5. **臨場反應與提問技巧：** 觀察顧問如何引導對話，而不是被動回答問題。
- ６大轉折重點
    
    **這通電話可以拆解為 6 個階段，請在聽錄音時，留意每個階段的轉折與顧問的用詞情緒。**
    
    | **階段 (時間戳參考)** | **顧問的任務目標** | **關鍵技巧與金句** |
    | --- | --- | --- |
    | **1. 破冰與資格確認 (00:00 - 01:20)** | - 喚醒記憶，建立連結
    - 確認基本資料 (年級、科系）- 探索程式語言基礎 | **提問技巧：** 從開放式問題開始 ("平常都用什麼語言？")，而不是封閉式問題 ("你會不會Python？")。 |
    | **2. 需求挖掘與目標釐清 (01:21 - 04:25)** | - 確認未來職涯方向 (求職、轉職- 鎖定理想職位 (資料科學家)
    - 找出當前學習瓶頸 (機器學習) | **顧問角色：** 「那你自己平常...」「你未來想往...」不斷將話題聚焦在「學生自身」。 |
    | **3. 痛點放大與價值塑造 (04:26 - 08:18)** | - 拋出「求職現實面」(要求工作經驗)
    - 強調「作品集」的重要性
    - 對比「學校」與「業界」的差距 (學用落差)
    - 描繪高薪願景，打破學生認知 (起薪 6 萬) | **金句製造機：**「面試官怎麼知道誰才是人才？」<br>「業界要的是即戰力」「產學落差的問題」 |
    | **4. 方案提出與實作導向 (08:19 - 14:11)** | - 將課程包裝成「實戰專案」與「作品集產出」
    - 強調「業界導師」的權威性 (PyCon 案例)
    - 將學習與「接案賺錢」連結，提升吸引力
    - 將「實作導向」與學生偏好連結 | **價值疊加：** 不只教技術，還教「溝通、議價、簽約」，這是超越所有競品的獨特賣點。 |
    | **5. 處理反對意見 (價格與時間) (14:12 - 16:55)** | - 將「價格」轉化為「投資」
    - 主動提出「學生分期方案」，降低壓力- 將「時間」問題轉化為「彈性規劃」與「投資回報」
    - 約定一對一深度諮詢，而非普通試聽 | **同理心展現：** 「我完全理解，大學生蠟燭兩頭燒...」先表示理解，再提供解方。 |
    | **6. 締結與下一步行動 (16:56 - 通話結束)** | - 鎖定下一次溝通時間 (明天七點)
    - 透過 LINE 建立後續聯繫管道- 指派「小作業」(選擇領域)，建立參與感與承諾 | **專業收尾：** 不是說「掰掰」就結束，而是清晰地告知對方接下來要做什麼，掌握主導權。 |
    
- 知識內容
    
    顧問在這通電話中提到了很多技術術語，把它們「翻譯」成客戶能理解的「利益」
    
    | **技術術語 (What it is)** | **顧問的比喻/翻譯 (What it means for you)** | **帶來的效果** |
    | --- | --- | --- |
    | **Python / C 語言 / Java** | 🗣️「Python **一行**就解決了，C 語言可能要寫**十幾行**。」 | **【好處】:** 效率高，省時間。讓客戶覺得學 Python 是更聰明的選擇。 |
    | **機器學習 / 爬蟲 / 數據分析** | (未深入解釋，而是將其包裝成...)<br>🗣️「讓你成為**資料科學家**的必備技能。」 | **【好處】:** 不是在賣技術，而是在賣一個「夢想職位」。將複雜技術與客戶的職涯目標掛鉤。 |
    | **Kaggle 競賽 vs. 業界專案** | 🗣️「競賽是**為了贏**，業界專案是**為了解決商業問題**。」 | **【價值差異】:** subtly 降低競賽的價值，提升「真實商業專案」的含金量，進而凸顯課程的獨特性。 |
    | **作品集 (Portfolio)** | 🗣️「證明你能力的證據」、「職場的**高鐵票**」、「跟別人不一樣的**履歷亮點**。」 | **【價值】:** 將一個抽象概念具象化。作品集不再只是一個檔案夾，而是通往成功的高速通行證。 |
    | **PyCon (Python Conference)** | (沒直接解釋縮寫)<br>🗣️「一個**大神級的業師**，代表台灣去國外跟全世界的人分享 Python。」 | **【權威感】:** 客戶不需要懂 PyCon 的細節，只需要知道「我們的老師很牛」。用故事和成就來塑造專家形象。 |
    | **接案 (Freelancing)** | 🗣️「賺到你**第一筆專案收入**」、「一個案子**八萬多**」、「讓你月入多**一萬到五萬**。」 | **【直接利益】:** 將學習的成果直接轉化為「錢」。這對學生來說極具吸引力，讓學費看起來像是可以快速回本的投資。 |
- 聽後練習（role play）
    1. **熟讀本指南：** 在腦中建立起這通電話的「骨架」如列點所看到Step1.2.3.4…
    2. **聆聽錄音檔 (至少 2 遍)：**
        - **第一遍：** 不用看稿，專心聽顧問的 **語氣、節奏和同理心**。感受他如何建立信任感。
        - **第二遍：** 對照逐字稿，找出每個階段的「關鍵提問」和「轉折句」，並在稿上做記號。
    3. **思考練習：**
        - 如果學生在「痛點放大」階段反駁你，說「我覺得學校教的夠用了」，你會如何回應？
        - 如果學生一開始就問「課程多少錢」，你會怎麼應對，並將話題拉回「價值討論」？
    4. **準備對練：** 選擇一個角色（顧問或學生），準備好與你的夥伴進行模擬演練。
