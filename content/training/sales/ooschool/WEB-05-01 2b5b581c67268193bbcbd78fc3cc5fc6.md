# WEB-05-01

ABC 分類: C
是否可接案: O
推薦話術: 此專案價值同 互動式動態網頁（後端），為進階版專案內容
適合對象: 後端工程師、雲端工程師
驗收標準: 熟悉後端框架的使用
能夠獨立完成資料庫表格設計
了解 API 設計與前後端分離概念
導入雲端部署服務
（選）能夠導入前端框架進行開發
預期完成時間: 6 個月
來源系列: 各領域專案_24.9.9
職能領域: WEB
專案描述: 1. 專案簡易解釋: 這是一個互動式動態網頁專案，包含CRM管理系統、電商網站和內容平台，屬於進階版的後端專案。

2. 主要內容在做什麼: 專案主要是開發一個能夠動態互動的網站，讓使用者能夠有效管理客戶資料、進行網上購物及內容發布。

3. 實際與市場銜接的技能和職缺、接案有哪些: 需要後端工程師和雲端工程師，技能包括熟悉後端框架、資料庫設計、API設計及雲端部署。接案機會包括為企業開發定制化的CRM系統或電商平台，能夠提升求職者在市場上的競爭力，學習成果包括能獨立完成專案及掌握最新的雲端技術。
