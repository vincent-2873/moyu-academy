# WEB-04-01

專案名稱: 互動式動態網頁（後端）
ABC 分類: C
是否可接案: O
推薦話術: 此專案價值同 多頁式靜態網頁（官方網站、電商前台），
為進階版專案內容，主要網頁增加了第三方API 串接，導入資料後，
後台的設計方式
適合對象: 後端工程師
驗收標準: 後端框架的使用
關聯式資料庫設計
（選）信件寄送功能、簡訊寄送功能
（選）購物車、訂單管理、金流與物流串接
（選）第三方登入 API 串接
預期完成時間: 3 個月
來源系列: 各領域專案_24.9.9
職能領域: WEB
專案描述: 1. 專案簡易解釋: 這是一個針對後端工程師的互動式動態網頁專案，主要是進階版的網頁開發，包含了第三方API的串接。

2. 主要內容在做什麼: 專案的重點在於設計後端系統，使用後端框架來處理資料，並設計關聯式資料庫。還可以選擇加入信件寄送、簡訊寄送、購物車、訂單管理等功能。

3. 實際與市場銜接的技能和職缺、接案有哪些: 這個專案需要的技能包括後端框架的使用、資料庫設計和API串接等。適合的職缺有後端工程師，接案方面可以接各類型的網站開發專案，特別是需要動態功能和資料處理的網站。

補充: 學習到的成果包括能夠獨立設計和實現一個完整的後端系統，並能夠與前端進行良好的互動，這對未來的職業發展非常有幫助。
