# WEB-02-01

專案名稱: 多頁式靜態網頁（官方網站、電商前台）
ABC 分類: B
是否可接案: O
推薦話術: 1. 多元化內容展示：多頁式靜態網頁可以展示更多元化的內容，包括公司介紹、產品或服務詳細信息、最新消息、常見問題等，讓訪問者更全面地了解您的業務。

2. 優化用戶導航：多頁式靜態網頁可以更有效地組織和管理內容，使訪問者更容易找到所需信息，提高用戶體驗。

3. 提升品牌形象：官方網站和電商前台是企業展示品牌形象和專業形象的重要平台，通過設計精美的頁面和引人注目的內容，可以提升品牌的知名度和認可度。

4. 增加互動和參與：在多頁式靜態網頁中，您可以添加各種互動元素，如表格、評論區、問卷調查等，促進訪問者的參與和互動，提高用戶黏性和參與度。

5. 支持SEO優化：每個頁面都可以針對特定關鍵詞進行優化，使您的網站在搜索引擎上更容易被找到，提高排名和流量。

6. 支持擴展和增量更新：多頁式靜態網頁可以根據業務需求進行擴展和增量更新，隨著業務的發展和變化不斷完善和更新內容。
適合對象: 網頁開發工程師、前端工程師
驗收標準: 表單設計與使用
前端套件使用（如 Chakra, Antd 等）
（選）CSR/SSR 差別與前端框架的使用
預期完成時間: 3 個月
來源系列: 各領域專案_24.9.9
職能領域: WEB
專案描述: 1. 專案簡易解釋：這是一個多頁式靜態網頁的專案，主要包括官方網站和電商前台，旨在展示企業的多元化內容。

2. 主要內容在做什麼：專案的內容包括公司介紹、產品或服務詳細信息、最新消息等，並透過優化用戶導航和提升品牌形象來改善用戶體驗。

3. 實際與市場銜接的技能和職缺、接案有哪些：適合的職缺包括網頁開發工程師和前端工程師，技能包括表單設計、前端套件使用（如Chakra、Antd）以及CSR/SSR的理解。接案機會主要集中在企業網站和電商平台的開發上。

補充：學習成果包括能夠設計和開發多頁式靜態網頁，提升網站的SEO優化能力，並能夠根據業務需求進行內容的擴展和更新。
