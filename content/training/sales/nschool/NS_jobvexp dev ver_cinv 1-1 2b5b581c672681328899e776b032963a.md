- 主要聽取內容重點
    1. **語氣與節奏：** 感受顧問溫和、自信且不急不躁的語氣。
    2. **提問技巧：** 注意顧問如何使用開放式問題，引導客戶自己說出困境。
    3. **同理與肯定：** 聽顧問如何先肯定客戶的努力，再點出知識缺口，讓對話氛圍保持正面。
    4. **化繁為簡：** 專注於顧問解釋專業術語的比喻和方式。
    5. **價值塑造：** 辨識出顧問是如何將「一小時諮詢」塑造為一個珍貴且能解決問題的機會。
- ６大轉折重點
    
    **這通電話可以拆解為 6 個階段，請在聽錄音時，留意每個階段的轉折與顧問的用詞情緒。**
    
    | 階段 (時間戳參考) | 顧問的任務目標 | 關鍵技巧與金句 |
    | --- | --- | --- |
    | 階段一：破冰與資格確認（客戶回復）(開頭 ~ 0:30) | 快速建立友善連結，並確認客戶是「有初步經驗但未系統學習」的目標客群。 | 技巧： 親切自然的開場，不帶銷售壓迫感。（客戶回復）金句： 「你已經有任何的投資理-財學習經驗了嗎？」 (一個簡單問題，直接進行客戶分層) |
    | 階段二：挖掘現況與痛點（客戶回復）(0:31 ~ 2:06) | 引導客戶自己陳述投資行為，並讓他意識到決策過程中的「不確定性」和「知識盲點」。 | 技巧： 連續使用開放式問題，讓客戶自己意識到問題的嚴重性。（客戶回復）金句： 「那你當時在買這些台股，你怎麼知道要選擇哪些標的呀？」(直接命中客戶決策依賴他人的痛點)（客戶回復）「那你有看過你的年化率目前大概是幾percent嗎？」(暴露客戶無法量化績效的盲點) |

- 知識內容
    
    顧問在這通電話中提到了很多技術術語，把它們「翻譯」成客戶能理解的「利益」
    
    | 技術術語 | 它的真實涵義 (What it is) | 顧問的比喻/翻譯 (What it means for you) |  |
    | --- | --- | --- | --- |
    | 年化報酬率 | 將不同時間長度的投資回報，換算成「平均每年」的回報率，用來比較不同投資的績效。 | 「你一年下來，平均是賺還是賠幾趴？」 |  |
    | ETF內扣費用 | ETF在管理過程中會產生管理費、交易費等，這些費用會直接從基金淨值中扣除，投資人不易察覺。 | 「這些費用其實都是內扣的，所以你會看不出來...你到底被扣了幾趴，你不知道。」 |  |
    | 時間成本 | 投入在沒有效益或效益低下的投資上的時間，錯失了將這段時間用在更高回報投資上的機會。 | 假設第一年賺10%，第二年賠10%。顧問說明：除了本金倒賠，還「整整賠掉兩年的時間」。 |  |
    | 時間複利 | 本金和利息一同再生利息，俗稱「利滾利」。時間越長，效果越顯著。 | 「它就是像滾雪球一樣，越滾越大。」（客戶回復）「第一年賺20萬...下一年是用120萬去賺20%，等於你是賺24萬。」 |  |
    | 三大面向分析（客戶回復）(基本面/籌碼面/技術面) | 分析股票的三種主流方法：（客戶回復）1. 基本面：看公司財報，評估其健康狀況。（客戶回復）2. 籌碼面：追蹤大戶（法人、主力）的買賣動向。（客戶回復）3. 技術面：研究價格走勢圖（K線），預測未來趨勢。 | 顧問直接用白話翻譯：（客戶回復）1. 基本面 → 公司賺不賺錢的「財務報表」。（客戶回復）2. 籌碼面 → 「大咖們」(三大法人、主力)在買還是賣。（客戶回復）3. 技術面 → 看價格圖(「K線」)，預判漲跌。 |  |
    | 止盈/止損 | 預先設定的賣出點。（客戶回復）1. 止盈：漲到目標價位就賣出，鎖定利潤。（客戶回復）2. 止損：跌到可承受的虧損點就賣出，避免更大損失。 | 顧問用一個生動的情境提問：「它已經漲到15%了，那我到底要賣嗎？還是我要繼續持有？」 |  |
- 聽後練習（role play）
    
    **情境一：扮演客戶**
    
    你是一位30歲的工程師，對投資有興趣，有在網路上看一些的資訊，並小額投入了約10萬元，目前小賺。你的痛點是：
    
    1. 市場波動太大，每天都很焦慮。
    2. 網路資訊真假難辨，不知道該信誰。
    3. 朋友都在討論股票，你覺得自己好像走錯了路。
    4. 你想學習，但工作太忙，沒時間研究。
    
    **情境二：扮演顧問 (由新人業務扮演)**
    
    你的任務是與這位客戶進行通話。請依照以下檢查清單完成你的任務：
    
    - **階段一：** 引導他說出上述至少 **2個** 痛點 (如：焦慮、資訊混亂)。
    - **階段二：** 針對他對加密貨幣的熱情，溫和地引入「資產配置」與「風險分散」的概念，建立你的專業性。
    - **階段三：** 同理他「工作忙、沒時間研究」的困境，並讓他產生「需要更有效率學習方法」的需求。
    - **階段四：** 順勢提出「一小時的顧問諮詢」，強調這能幫他「建立一套適合自己的投資系統，而不是追逐風口」。
    - **階段五：** 成功敲定諮詢時間，並處理好所有細節。
