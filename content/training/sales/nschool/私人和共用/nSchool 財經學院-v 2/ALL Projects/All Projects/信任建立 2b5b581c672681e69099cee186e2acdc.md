| **信任建立階段** |  | 行為/心理狀態演變 |
| --- | --- | --- |
| **目標** | 用數據、案例和同理心消除客戶對課程的抗拒心理。 |  |
|  | 讓客戶感受到學習的必要性與課程的價值。 |  |
| **應注意** | 請多用【你】【我】【他】的方式來進行第三方說詞，更能讓客戶產生出第三方客觀視角來補充說明 |  |
| **為什麼這樣做** | 成功案例能讓客戶產生共鳴，覺得課程與自身的經驗相關。 |  |
|  | 用一些數據支撐(複利)提高可信度，降低客戶對課程效果的質疑，讓客戶認為你是有料的。 |  |
|  | 複利效應的具體數字展示，讓學習的價值更加具體化。 |  |
| **話術/實例模擬-建立專業信任** | 像你提到的問題，以我們學員來說就是能幫助他快速掌握選股技巧。例如，如何用毛利率篩選出具競爭力的企業。 | 信任感基礎開始，專業程度上升➡️舉例、比喻、套用、說明 |
| **話術/實例模擬-分享成功案例** | 跟你分享我之前一位學員，他和你一樣(靠朋友推薦買股票)((看他情境式什麼，就是跟他一樣就好)，但透過我們教育學習的方式，學會用營收成長率篩選標的，現在每年穩定增長10%，你知道每年穩定增長10%是甚麼概念嗎?哈哈 | 帶入你我他的客觀意識來去做說明，增加客戶客觀主體印象➡️並且帶入客戶自己本身的經驗和經歷(如果沒有就說股市詐騙的事情和入群的事情)➡️反問方式來帶入確認客戶是否對數值有概念，並且為下一句增加一些鋪陳和產品信心資訊 |
| **話術/實例模擬-解釋複利效應** | (客戶知道)你很讚ㄟ！你說的蠻清楚的　(客戶不知道) 沒關係，我的學生(剛剛舉例你我他的例子來進行衍伸)假設每年能穩定達到10%的回報，10萬元在30年後會變成174萬元，這就是知識與資金結合產生的復利效果。 | 認同及讚美，讓客戶覺得你跟他是同一線➡️用你我他的方式來進行客觀事實加強➡️後續解釋及說明產品知識和實務上是有辦法結合的 |
| **話術/實例模擬-引發學習興趣** | 我們不只是教理論，還會提供實操APP，和投資節點去做教學，主要要讓每一位學員把這些知識應用到自己平常生活上的投資中。 | 說明為何我們可以讓學習的客戶進行有成果，並且說明是大概怎麼樣式使用到這些資訊 |
| **話術/實例模擬-具體的複利價值延伸** | 很多人只看短期收益，卻忽略了長期的複利增長。比如說，您每月投資5000元，假設年化報酬率10%，30年後資金會成長到超過1130萬元，這就是複利的驚人力量，你這樣思考看看，一個月5000你這樣剛剛聽我這樣說，是不是當時你自己投入的成本都還比現在多更多哈哈 | 說明真正投資學習的重要性和複利的成果，尾端針對現有的學習內容和方式來去加強客戶對於5000原來說一個月是小錢 |
| **話術/實例模擬-應用情境與客戶反應** | 客戶可能回應：「聽起來不錯，但我平常比較忙，擔心學不會。」 |  |
|  | 回應話術：「在我們學習上其實非常靈活，不管你是在台灣還是被綁去東南亞你只要連上線就可以去做線上學習，跟你在下單投資一樣，我們主要也是客製化，所以你學到的東西你會了我們不會重複性交給你讓你去做學習，我們您可以根據自己的時間進行學習，確保每一步都能掌握。」 | 自嘲➡️並說明產品優點 |
| **話術/實例模擬-更深層次的複利探討** | 你知道嗎？即使是每年8%的穩定增長，也能讓10萬元在20年後翻倍兩次，達到接近50萬元。重點是穩定和持續。 | 選擇當下是否要說 |
