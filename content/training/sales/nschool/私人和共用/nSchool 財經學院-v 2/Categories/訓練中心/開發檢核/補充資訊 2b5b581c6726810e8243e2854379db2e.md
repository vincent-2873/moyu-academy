# 補充資訊

文字: 新人開發需聚焦於客戶需求與預算，避免過早介紹自己以建立信任，並透過衍生話題來引導客戶對投資學習的興趣。重要的是掌握通話的起承轉合，增進客戶信任並邀約進一步交流。

| **新人前期開發，請幫我先針對這兩點先做資料蒐集** |
| --- |
| 1. 【需求】->客戶經驗、我們的產品能不能滿足她去做評估 |
| 2. 【預算】->業務最終銷售終點，是否有預算學習金額來去投入學習 |
|  |
| **開發注意事項** |
| 1. 要去懂得避開和後續引導 |
| 避開->不代表逃避、不回應，而是要讓客戶重點不在這身上 |
| ex. 客戶問問題，是要專注在【她對這領域興趣】不是【為甚麼我今天會接到你的電話】 |
|  |
| 2. 要懂得衍生性話題 |
| 懂得去了解應用所說的，今天在學習的時候，如何識別哪些是所謂【開話題專用】那些是【銷售必備蒐集利器】 |
| 應用客戶她當下說的內容，來去衍生說明和話題衍生並認同客戶 |
| EX. 客戶講到她有投資基金 |
| 第一反應->認同、鼓勵 你真的很厲害欸，那你基金怎麼挑選標的->衍生詢問，再來詢問怎麼學習 |
| 自己可以針對每一個話題來去練習話心智圖和架構，遇到甚麼樣的客戶，面對什麼樣的話題該怎麼接球並且反問客戶釣出客戶痛點並回歸到我們股市投資學習的話題主軸上 |
|  |
| 3. 別太急著於介紹【我是誰】 |
| 今天一通電話過去，不是說開不到30秒就要介紹我是誰，而是今天要先跟她取得所謂【信任】 |
|  |
| 複習一下今天教的信任度提升方法 |
| 1. 【我】的親身經歷 |
| 2. 【我朋友】【我的學員】【我家人】的過程 |
| 用客觀的角度來去更加驗證你說的每一句話，並且加深客戶讓她有一個印象會是，雖然話都是你說的但是你多是用例子，主觀會讓人盲目，並且容易會維護自己的主觀意識而去有情緒、無法理解對方再溝通甚麼 |
|  |
| 那對於別太急著介紹【我是誰】主要是因為 |
| 1. 太早說出，別人就會對你失去興趣 |
| 2. 許多事情需要事先鋪陳，後續再讓他們【主動】詢問說【所以...你們是甚麼?】讓客戶自然連結我們與和她前面溝通她自己本身過往投資經歷、投資學習等等之類的事情，要再重要的時候再介紹我們把前後兩者相呼應起來 |
|  |
| 電銷最重要的也就是起承轉合 |
| 起->破冰、開頭、詢問、印象 |
| 承->對於事件的鋪承、客戶資料蒐集、客戶信任疊加 |
| 轉->回歸到自己身上，並且介紹自己，且和【承】的時候有相互呼應 |
| 合->收斂前面討論的事情，並回歸於【投資學習】身上，邀約demo，加Line |
