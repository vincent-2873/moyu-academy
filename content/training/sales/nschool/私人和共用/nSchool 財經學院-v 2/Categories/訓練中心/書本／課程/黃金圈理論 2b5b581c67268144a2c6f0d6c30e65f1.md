# 黃金圈理論

Created time: 2025年11月24日 上午8:27
課程介紹: 這是一門針對電話銷售、諮詢顧問、新人業務所設計的對話邏輯訓練課，核心教學基礎為 Simon Sinek 提出的 「黃金圈理論 Golden Circle」，透過【WHY → HOW → WHAT】三層架構，搭配 SPIN 話術，幫助業務從說服變成「引導」，從話術變成「洞察」。
狀態: 已完成

### 💡 課程目的／要幹嘛？

✅ 協助業務理解「成交是從動機開始」的原則

✅ 建立有邏輯、有情緒、有信任的對話節奏

✅ 能將話術套用於實際產品：ETF、AI交易、作品集設計、剪輯等

✅ 提升開場3分鐘內吸引注意力與促進回應的能力

---

### 🧩 教學模組結構

| 單元 | 主題 | 說明 |
| --- | --- | --- |
| 1️⃣ | 無效話術的常見陷阱 | 說明為何多數新手容易跳過 WHY，造成「聽完沒感覺」的溝通落點 |
| 2️⃣ | 黃金圈三層結構 | 拆解 WHY/HOW/WHAT 的心智與心理影響邏輯 |
| 3️⃣ | 財經話術案例練習 | ETF、顧問、模擬倉的完整 WHY→HOW→WHAT 示範 |
| 4️⃣ | 職能話術案例練習 | UIUX、設計、剪輯等跨領域課程如何套用話術架構 |
| 5️⃣ | SPIN × 黃金圈整合技術 | 對應四階段問句（現況→問題→擴大→行動）與黃金圈的關係 |
| 6️⃣ | 小白SOP話術模板 | 3步驟開場話術，適用初學者一開口就能用 |
| 7️⃣ | 30秒成交關鍵設計 | 讓學員練習在30秒內建立信任＋行動框架 |
| 8️⃣ | 實戰模擬演練 | 角色扮演練習：一人業務／一人客戶／一人觀察指導 |

### 🎯 本課適合對象

- 初入職場或轉職到【財經】或【職能】領域的銷售新手
- 常遇到「講完對方沒反應」的業務
- 想提升話術邏輯、客戶共鳴與成交能力者
- 業務主管想訓練團隊具備「有邏輯感的溝通能力」

---

### 🛠️ 上完這堂課，你會學到：

- 如何用「Why」打動人心，而不是一開始就講產品（What）
- 如何讓對方說出自己想要改變，而不是你一直說服
- 如何將【話術】轉化為【客戶認同感】與【主動行動】
- 如何快速帶出產品特色又不被當推銷員拒絕
- 實際用在電話開場、Demo 說明、引導成交、社群私訊開場等場景

---

### 🧪 實用應用場景範例（含財經／職能）

| 情境 | 話術應用（簡略） |
| --- | --- |
| 誘因不足 → 不想聽你說 | 「你覺得現在投資的方式，真的讓你安心嗎？」 |
| 已嘗試其他方案 → 無感 | 「之前上過的課，都卡在哪個地方？」 |
| 覺得自己學不會 → 沒信心 | 「很多人都這樣覺得，但當開始有成果，就回不去了。」 |
| 不知道如何開始 | 「我們可以先從我們這次學習先開始，你邊看邊操作。」 |
