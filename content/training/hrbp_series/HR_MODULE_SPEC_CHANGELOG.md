- 「使用者是人」
- 「有比較就會有傷害」
- 「知道→做到→做好→精通」

## HR-056 致命提問應對 · 業績壓力與團隊氛圍

|------|------|
| 單元代號 | HR-056 |
| 所屬體系 | HR |

| 優先級 | P0 |
| 影片時長 | 11 分鐘 |
| 互動測驗 | 8 題（通過門檻 6/8） |
| 前置單元 | HR-053、HR-054、HR-055 |
| 系列位置 | 招募訓練系列 EP4/4（系列完結） |
| 核心產出 | 影片（阿凱+Vincent 原聲）+ 互動測驗 + 系列完結儀式 |

**學習目標**：
1. 在業績壓力提問時，不拐彎不推銷地正面回應
2. 在團隊氛圍問題上用事實（非形容詞）描繪
3. 避免「曖昧不清」的回答
4. 用教育訓練制度給求職者安心感

**核心金句**：
- 「曖昧不清的關係沒人喜歡」
- 「有，一定會有業績壓力」
- 「我們不丟你自己扛」
