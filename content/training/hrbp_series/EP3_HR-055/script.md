## 第 2 段（1:30-3:00）心法的原點：使用者是人

**[切 Vincent 原聲 V3-04]**  
> *「我們不只是問問題,我更希望可以往下進行多說一點,畢竟使用者是『人』。人與人之間需要知道：你的需求是什麼？為什麼想做這件事？我該怎麼幫助到你？」*

**[阿凱]**  
「聽 Vincent 這段。關鍵字是 —— **使用者是人**。」

**[畫面]** 兩個簡筆人像面對面

**[阿凱]**  
「你招募不是填表格，是在跟『一個想換工作的活生生的人』對話。  
他有焦慮、有期待、有怕被騙、有好奇。你要做的不是問完 20 題打分數，是**理解他然後告訴他你能幫什麼**。」

**[圖卡]**
```
心法原點
─────────
不是「問到答案」,
是「理解後協助」
```

**[阿凱]**  
「記住這個原點。接下來所有技巧，都是從這裡長出來的。」

## 第 3 段（3:00-6:00）三階段回覆法 · 完整結構

**[畫面]** 三個階段的階梯圖從下往上

**[切 Vincent 原聲 V3-01]**  
> *「我們有時候可以做一些鋪陳。有三階段的回覆：第一階段：標準回覆，比較公版跟知識的內容。第二階段：延伸，帶入實際狀況跟例子、產業資訊來鋪陳陳述。第三階段：反問求職者，他自己是怎麼想的。」*

**[阿凱]**  
「這段要聽三次。Vincent 這個三階段法，是整個招募對話的**底層框架**。」

**[大型圖卡]**
```
🥇 第一階段：標準回覆
   → 公版知識,先把事實講清楚

🥈 第二階段：延伸
   → 帶入例子、產業資訊、真實狀況

🥉 第三階段：反問
   → 把球丟回給求職者,問他怎麼想
```

**[阿凱]**  
「這三階段不只是說話順序，是**資訊流動的節奏**：」

**[畫面]** 三個箭頭：你→他→你→他→平衡

**[阿凱]**  
「標準 = 你給他資訊  
延伸 = 你加值資訊  
反問 = 他給你資訊  
整個對話變成**雙向流動**，不是你單方面播音。」

## 第 4 段（6:00-8:30）實例拆解 · 團隊氛圍怎麼講?

**[畫面]** 案例標題出現：「求職者問:團隊氛圍怎麼樣?」

**[阿凱]**  
「用一個最常見的題目示範。求職者問『你們團隊氛圍怎麼樣？』，這題很多新人會答爆。」

**[阿凱]**  
「**錯誤回法**：『我們團隊很好啊、很 nice、大家都很好相處。』  
聽起來沒錯，但求職者會覺得你在敷衍。為什麼？因為**沒有內容、沒有畫面、沒有反問**。」

**[切 Vincent 原聲 V3-02]**  
> *「我們的團隊氣氛並不是說很壓迫,但是節奏很快,大家都在做事也會相互幫忙。業績壓力勢必會有,因為這是業務,但我們是有節奏地在推,不會是被罵的那種。你願意做、願意學,我們也會幫你一起把業績跟目標做出來,而不是丟給你自己去扛。」*

**[阿凱]**  
「這是 Vincent 示範的**第一階段 標準回覆**。拆解一下為什麼好 —— 」

**[圖卡]**
```
Vincent 的標準回覆拆解
──────────────────
✓ 正面定調:「不是壓迫」
✓ 描述實況:「節奏快、互相幫忙」
✓ 誠實承認:「壓力一定有」
✓ 緩解焦慮:「不會是被罵的那種」
✓ 傳遞承諾:「幫你一起扛」
```

**[阿凱]**  
「注意到沒？這段話**講的是事實、不是形容詞**。  
『很好』是形容詞，『不會被罵、幫你扛』是事實。事實才有說服力。」

## 第 5 段（8:30-10:30）第二階段 · 延伸 · 有比較就會有傷害

**[畫面]** 天秤圖，兩邊放「傳統業務 vs 我們」

**[阿凱]**  
「第一階段把事實講完還不夠，進到**第二階段 延伸**。Vincent 有一招絕的 —— 」

**[切 Vincent 原聲 V3-03]**  
> *「延伸回覆則是秉持核心原則：有比較就會有傷害。在招募時怎麼做到這件事？我們要說到市場的業務。目前市場上很多傳統型業務完全沒有任何教育訓練流程。我們可以跟求職者說：我們是有別於以往傳統業務團隊（像是保險或傳統無制度化的團隊）。為什麼很多求職者進來後很喜歡我們的地方？因為我們有明確的教育訓練。」*

**[阿凱]**  
「『有比較就會有傷害』—— 這六個字是**業務招募的心法之王**。」

**[圖卡]**
```
延伸公式:用市場對比襯托我們
──────────────────────────
❌ 傳統業務:沒系統、靠自己摸
❌ 保險模式:無底薪、高壓

✅ 我們:固定課綱、學長姊帶、帶得走
```

**[阿凱]**  
「把這個框架記下來。當你講『我們有教育訓練』時，求職者反應平淡；  
但如果你講『**市場上大部分業務是丟你自己去撞，我們不是**』，求職者耳朵會豎起來。」

**[阿凱]**  
「為什麼？因為**單講優點沒感覺，有比較才有感覺**。這是人性。」

## 第 6 段（10:30-12:00）第三階段 · 反問 · 丟球回去

**[畫面]** 籃球來回傳遞的動畫

**[阿凱]**  
「最後一階段 —— **反問**。這一階段 90% 的新人會漏掉，但這是讓對話**真正變成對話**的關鍵。」

**[圖卡：反問金句庫]**
```
💬 反問萬用句
─────────────
・「那你覺得呢?」
・「你之前待的團隊氛圍是怎樣?」
・「對你來說,好的團隊長什麼樣?」
・「聽完這段,你有什麼疑問?」
・「你在意的是壓力,還是成長速度?」
```

**[阿凱]**  
「反問有三個好處 —— 」

**[圖卡]**
```
① 讓對方說話:你才能聽到他真正的想法
② 展現尊重:你不是單方面推銷
③ 蒐集情報:他的答案決定你下一步話術
```

**[阿凱]**  
「⚠️ 特別提醒：**『你還有想問的嗎？』不是好的反問**，這句話容易讓對話 dead end。  
改成『你剛剛有提到 XX，我想多聊一下 —— 』，才是好的丟球。」

**[畫面]** 四階段階梯動畫

**[阿凱]**  
「最後講成長。Vincent 有一句話我特別喜歡 —— 」

**[切 Vincent 原聲 V3-05 或圖卡]**  
> *「一開始電訪或一面的時候,我們一定會先讓你們了解流程,形成肌肉記憶。」*

**[圖卡：肌肉記憶四階段]**
```
🎯 知道 → 做到 → 做好 → 精通

1. 知道:看過、聽過、記得這個概念
2. 做到:照著做,結果符合標準
3. 做好:做起來自然、不卡、有彈性
4. 精通:有自己的版本,能教別人
```

**[阿凱]**  
「你現在看這支影片是『**知道**』。  
實際打第一通電話是『**做到**』。  
打 50 通之後開始順了是『**做好**』。  
有天你能跟新人解釋『為什麼這樣講』是『**精通**』。」

**[阿凱]**  
「別急，沒有人一開始就精通。」

**[阿凱]**

**[圖卡]**
```
拒絕心法
─────────
・他不是拒絕你,是我們不合適
・雙向選擇,你也在篩他
・每一次拒絕都在校正你的判讀
```

**[阿凱]**  
「把拒絕當成資料點，不要當成否定。這是心理衛生，也是能在這行做久的關鍵。」
