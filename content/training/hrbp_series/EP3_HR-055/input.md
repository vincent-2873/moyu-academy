## 🎯 學習目標
1. 新進 HRBP 能熟練運用「標準 → 延伸 → 反問」的三階段回覆法。
2. 能在非結構式（拉近距離）與結構式（系統詢問）間自然切換。
3. 能從求職者履歷中找到職務共通點並延伸對話。
4. 能應用「肌肉記憶四階段」（知道→做到→做好→精通）檢視自己的成長。

## 🔑 關鍵知識點
- **三階段回覆法**（Vincent 核心心法）：
  - 第一階段：標準回覆（公版內容）
  - 第二階段：延伸（實例 / 產業資訊 / 鋪陳）
  - 第三階段：反問（丟球給求職者）
- 非結構式 → 結構式的訪談節奏
- 黃金圈：What / How / Why，挖到動機層
- 肌肉記憶：知道 → 做到 → 做好 → 精通
- 「丟球給求職者」的藝術：不只自己講
- 「有比較就會有傷害」—— 用市場對比襯托我們
- 被求職者拒絕也 OK 的心態
