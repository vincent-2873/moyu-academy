# HR-054 一面 vs 二面：面試的任務分工

- **所屬體系**：HR

- **優先級**：P0
- **預估總時長**：影片 11 分鐘 + 互動測驗 7 分鐘

- **系列位置**：HRBP 招募訓練系列 EP2/4

## 🎯 學習目標
1. 新進 HRBP 能明確區分一面與二面的任務目標。
2. 能根據「應聘條件表」判斷一面哪些特質該過濾。
3. 能避免「一面二面講一樣東西」的常見失敗。
4. 能理解「一面預測未來績效」的思維方式。

## 🔑 關鍵知識點
- 一面負責：公司/集團/品牌/制度介紹、求職者適配性判斷
- 二面負責：主管魅力、願景、價值觀、心態溝通
- 一面的判斷關鍵：是否主動詢問、對制度是否想多了解
- 進階能力：從一面預測錄取後的績效
- 常見錯誤：一面跟二面內容重複 → 浪費彼此時間
- 應聘條件是二面主管訂的，一面要過濾基礎門檻
- 「勇於試錯」「勇於承擔」vs「主觀意識太強」的判讀

## 🎬 情境素材
**情境 1：新 HRBP 不知道什麼時候該推進二面**
- 來源：逐字稿 2，芸蓁問「怎麼判斷只要一面還是需要到二面」
- 解答：看應聘條件 + 適配感

**情境 2：一面把二面的話講光了**
- 問題：二面主管沒話講，失去二面價值
- 正解：一面 = 硬知識；二面 = 軟連結

**情境 3：求職者面試表現平庸但「看起來乖」**
- 陷阱：可能只是「想了解」不是真的想進來
- 觀察：有沒有主動提問

## 📋 核心素材（Vincent 原話）
- 「一面我們大概會講什麼東西。不外乎公司的集團的介紹公司的介紹品牌的介紹還有團隊的規模型態」（逐字稿 1, 17:48）
- 「過往其他間招聘一面跟二面會很像，有點像是失去一面跟二面互相的價值」（逐字稿 1, 18:13）
- 「二面那可能像是...透過主管自己的一個我們講價值對或者說今天為什麼你要選擇為什麼是我要去帶著你一起去走這個願景」（逐字稿 1, 18:35）
- 「熟悉這個階段後，甚至能預測如果錄取，他未來的績效會是多少」（逐字稿 2, 05:29）
- 「怎樣叫『勇於試錯』、『勇於承擔』，或是『主觀意識太強』」（逐字稿 2, 06:23）
