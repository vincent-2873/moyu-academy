## 第 1 段（0:00-1:00）開場

**[阿凱]**  
「上一集你學會了怎麼跟求職者講薪水。但薪水講完之後呢？對方同意要面試了，然後你發現 —— 我們公司有**兩輪面試**。」

**[圖卡]** 一面 | 二面

**[阿凱]**  
「很多新 HRBP 會卡在這一關：『我到底該跟求職者聊什麼？什麼是一面要做、什麼是二面要做？』  
這不只是流程問題，是一個**更嚴重的陷阱** —— 如果你一面把二面的話講光了，二面主管就沒戲可唱。」

**[切 Vincent 原聲 V2-01]**  
> *「過往可能其他間招聘一面跟二面會很像。因為他沒有辦法去分接出來說今天一面要講什麼話，二面要講什麼，那有點像是失去一面跟二面互相的價值。」*

**[阿凱]**  
「這集 11 分鐘，我幫你把這兩輪面試的『任務分工』講清楚。」

## 第 2 段（1:00-2:30）核心對比圖

**[畫面]** 左右分割大圖卡

**[阿凱]**  
「先看全景。一面和二面，你要記住這張表 —— 」

**[大型對比圖卡]**
```
         一面                    二面
      ─────────              ─────────
角色    HRBP（你）             據點主管
任務    介紹 + 篩選            連結 + 邀請
內容    公司/品牌/制度         主管風格/願景/帶人方式
問題    他適不適合？           他要不要跟我走？
時長    約 30 分鐘             約 30-60 分鐘
關鍵    硬知識傳遞             軟連結建立
```

**[阿凱]**  
「最簡單的記法：**一面是『介紹』，二面是『邀請』**。」

**[阿凱]**  
「一面是帶求職者認識公司這個『環境』；二面是讓主管跟求職者建立『人對人』的連結。  
這兩件事職責不同、情緒目標不同、甚至說話的節奏都不同。」

**[畫面]** 四個圖卡依序浮現

① 公司 / 集團 / 品牌介紹

## 第 4 段（4:00-5:30）二面要留什麼給主管？

**[畫面]** 切到二面場景

**[切 Vincent 原聲 V2-02]**  
> *「二面那可能像是不管是營運還是說業務，透過我們講自己本身據點主管或者說主管自己的一個我們講價值對或者說今天為什麼你要選擇為什麼是我要去帶著你一起去走這個願景。」*

**[圖卡]**
```
二面主管會做的事（你 HRBP 不要搶）
────────────────────────
❌ 不要講「主管為什麼帶你」
❌ 不要講「我們團隊的願景」
❌ 不要講「主管怎麼訓練新人」
❌ 不要承諾「這個主管很好相處」
```

**[阿凱]**  
「這些話要**留給主管自己講**。你講了兩個問題 —— 第一，你講的沒主管本人講來得有力；第二，主管到二面沒話可補，求職者會覺得『那我為什麼還要二面？』」

**[切 Vincent 原聲 V2-03]**  
> *[受訓 HRBP 發言] 「二面有點像是去討論價值觀，就是適不適合這個團隊，然後心態的想法。也可以透過二面的時候，透過交談讓求職者對這份工作更有企圖心。」*

**[阿凱]**  
「這句總結非常準：**一面給資訊，二面給期待**。這就是兩輪面試不能互相取代的原因。」

## 第 5 段（5:30-7:30）一面最難的任務 · 判斷適配性

**[畫面]** 一張「應聘條件表」示意圖浮現

**[阿凱]**  
「回到一面的第四項任務 —— 判斷適配性。你怎麼知道這個人能不能通過二面？」

**[阿凱]**  
「答案在『**應聘條件表**』。這張表上，據點主管會列出他要找的人長什麼樣：」

**[圖卡範例]**
```
📋 應聘條件表（示意）
────────────────────
年齡     35 歲以下
經驗     有業務經驗尤佳
特質     主動積極、表達清晰
排除     主觀意識太強
加分     曾獨立開發客戶
```

**[阿凱]**  
「你一面的任務，就是**用這張表當標尺**。」

**[切 Vincent 原聲 V2-05]**  
> *「面試過程中一定會感受到：第一，面試者是否會主動詢問？對制度或公司想法是否想多了解？這些都看得出來。」*

**[阿凱]**  
「注意聽這段最重要的線索 —— **主動詢問**。有沒有主動問問題，是判斷適配性最簡單的指標。」

**[圖卡]**
```
✅ 加分訊號
   ・主動問制度細節
   ・主動問主管帶人風格
   ・對薪資計算有好奇心

❌ 減分訊號
   ・只回答你問的，不反問
   ・對制度沒反應
   ・眼神沒有光（聽到業績壓力反應平淡）
```

## 第 6 段（7:30-8:45）進階能力 · 從一面預測績效

**[畫面]** 水晶球 emoji 圖卡 + 數據圖表

**[切 Vincent 原聲 V2-04]**  
> *「熟悉這個階段後，甚至能預測如果錄取，他未來的績效會是多少，這是我們最後要達到的目標。但過程中一定會先透過了解一面的詢問方式，再來是判斷標準。」*

**[阿凱]**  
「聽到沒？**一面厲害的 HRBP，可以預測錄取後的績效**。這聽起來像魔法，其實是經驗累積。」

**[圖卡]**
```
預測績效的三個線索
──────────────
1. 他對「數字」的敏銳度
   → 算薪水會不會興奮？

2. 他對「拒絕」的承受度
   → 提到業績壓力會不會退縮？

3. 他對「學習」的飢餓感
   → 有沒有追問教育訓練細節？
```

**[阿凱]**  
「當這三題你都能答『yes』，這個人十之八九能做起來。這是 HRBP 的終極技能 —— 不是篩選，是**預見**。」

## 第 7 段（8:45-9:45）特質判讀 · 勇於試錯 vs 主觀意識太強

**[畫面]** 分割畫面，左邊綠色勾勾、右邊紅色叉叉

**[阿凱]**  
「最後講一個實戰上最常被問的 —— 怎麼分『勇於試錯』和『主觀意識太強』？這兩個有時候真的很像。」

**[圖卡對比]**
```
勇於試錯（✅ 錄取）    vs    主觀意識太強（❌ 過濾）
────────────              ────────────
「我覺得可以試試          「我之前就是這樣做
 ，錯了再改」              ，為什麼要改？」

願意先聽你說              打斷你講，堅持己見

承認錯誤是成長            把錯誤歸咎環境

跟主管磨合後能調整         跟主管磨合後更固執
```

**[阿凱]**  
「判讀方式：**在一面故意給他一個跟他認知不同的資訊，觀察他的反應**。」

**[阿凱]**  
「例如他說『保險業務都會給底薪』，你說『我們這邊職能有底薪，但財金是擇優領』，看他反應 —— 是好奇問『為什麼？』還是堅持『那怎麼會有人做？』。這一秒的差別，就是兩種人的分野。」

## 第 8 段（9:45-10:30）常見失敗 · 一面二面講一樣

**[畫面]** 警示燈閃爍

**[阿凱]**  
「結束前提醒你**最常見的失敗模式** —— 一面和二面講一樣的東西。」

**[圖卡]**
```
⚠️ 失敗案例

HRBP 一面：「我們主管很 nice，會手把手教你...」
主管二面：「我會手把手教你...」
求職者 OS：「所以我為什麼要花兩小時面試？」
```

**[阿凱]**  
「避免方法：**把『主管個人特質』列進心中黑名單**。你可以講團隊結構、訓練流程、升遷路徑，但不要預先描述主管的個性與風格 —— 那是主管二面要建立的第一印象，你一面講了就穿幫了。」
