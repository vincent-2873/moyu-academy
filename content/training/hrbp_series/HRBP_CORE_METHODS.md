## 🎯 心法一：三階段回覆法（對話底層框架）

遇到任何問題，套這個結構就不會答爆：

```
1️⃣ 標準回覆   → 公版知識，先把事實講清楚
2️⃣ 延伸        → 帶入例子、產業資訊、真實狀況
3️⃣ 反問        → 把球丟回給求職者，問他怎麼想
```

**為什麼有效**：  
這不只是說話順序，是**資訊流動的節奏**。標準 = 你給他、延伸 = 你加值、反問 = 他給你。整個對話變成雙向流動，不是你單方面播音。

---

## 🎯 心法二：曖昧不清沒人喜歡

> 「要嘛在一起，要嘛不要。」— Vincent

遇到致命提問（業績壓力 / 團隊氛圍 / 做不到會不會被 fire）時，**不能拐彎**。

**拐彎語言黑名單**：
- 「看你想領多少」
- 「每個人感受不同」
- 「可能會有一點」
- 「要看你怎麼定義」
- 「你自己來就知道了」

以上聽起來中性、其實把球踢回求職者，會讓對方感覺你在推銷或隱瞞。

---

## 🎯 心法三：有比較就會有傷害（延伸公式）

單講「我們很好」沒有說服力。用市場對比襯托我們 —— 

```
❌ 市場上傳統業務：丟你自己撞、沒課綱
❌ 保險模式：無底薪、高壓
✅ 我們：固定課綱、學長姊帶、帶得走
```

一句話：**沒有對比，優點沒感覺。**

---

## 🎯 心法四：使用者是人

> 「我們不只是問問題，我更希望可以往下進行多說一點，畢竟使用者是『人』。」— Vincent

你招募不是填表格，是在跟「一個想換工作的活生生的人」對話。  
→ 不是「問到答案」，是「理解後協助」

## 📞 公版話術庫

### 話術 A｜求職者問「我可以領多少？」

```
❌ 我們抽 6%-15%
✅ 光底薪 3 萬、光抽成大概 1-2 萬、
    活動獎金 6000-8000，加起來一個月 4-5 萬
```

**原則**：給 Range + 實拿金額，不報％。

---

### 話術 B｜求職者問「有沒有業績壓力？」（Vincent 黃金三句）

```
第一句：有，一定會有業績壓力（承認）
第二句：業務就是有業績壓力才叫業務（正常化）
第三句：重點是我們怎麼幫妳化解 —— 
        第一週固定課表、第二週學長姊帶單，
        不是丟妳自己扛（轉向支持）
```

---

### 話術 C｜求職者問「團隊氛圍怎麼樣？」（Vincent 示範版）

```
我們的團隊氣氛不是壓迫型，但節奏很快，
大家都在做事也會互相幫忙。

業績壓力勢必會有，因為這是業務，
但我們是有節奏地在推，不會是被罵的那種。

你願意做、願意學，我們也會幫你一起把業績
跟目標做出來，而不是丟給你自己去扛。
```

**拆解心法**：
- 定調（不是 X）+ 具象（而是 Y）
- 承認事實（壓力會有）
- 緩解恐懼（不是被罵）
- 承諾支持（幫你扛）
- **每句都是事實，不是形容詞**

---

### 話術 D｜求職者問「做不到會不會被 fire？」

```
我們是階梯式降級，一次只降一階，不會直接打回新人。
這是公司給業務的犯錯緩衝空間。

而且新人第一週有固定課表、第二週學長姊帶單，
你不是一個人在扛。
```

---

### 話術 E｜反問萬用句（第三階段）

```
・「那你覺得呢？」
・「你之前待的團隊氛圍是怎樣？」
・「對你來說，好的團隊長什麼樣？」
・「你在意的是壓力，還是成長速度？」
・「你剛剛有提到 XX，我想多聊一下 —— 」
```

**⚠️ 避免**：「你還有想問的嗎？」容易讓對話 dead end。

## 🔍 判讀指南

### 一面特質判讀 · 加減分訊號

| ✅ 加分 | ❌ 減分 |
|---------|---------|
| 主動問制度細節 | 只回答你問的 |
| 主動問主管帶人風格 | 對制度沒反應 |
| 對薪資計算有好奇心 | 聽到業績壓力反應平淡 |
| 承認錯誤是成長 | 把錯誤歸咎環境 |
| 願意先聽再說 | 打斷你講，堅持己見 |

---

### 勇於試錯 vs 主觀意識太強

```
勇於試錯 (✅ 錄取)          主觀意識太強 (❌ 過濾)
──────────────         ──────────────
「可以試試，錯了再改」       「我之前就是這樣做」
願意先聽你說                打斷你講，堅持己見
承認錯誤                    歸咎環境
跟主管磨合能調整              跟主管磨合更固執
```

**判讀方式**：故意給一個跟他認知不同的資訊，觀察反應是「好奇問為什麼」還是「堅持那怎麼會有人做」。

---

### 績效預測三線索（進階）

```
1. 對「數字」的敏銳度   → 算薪水會不會興奮？
2. 對「拒絕」的承受度   → 提業績壓力會不會退縮？
3. 對「學習」的飢餓感   → 有沒有追問訓練細節？
```

三題全 Yes → 十之八九能做起來。

## 🎚️ 一面 vs 二面 快速對照

|  | 一面 | 二面 |
|--|------|------|
| 角色 | HRBP（你） | 據點主管 |
| 任務 | 介紹 + 篩選 | 連結 + 邀請 |
| 內容 | 公司/品牌/制度 | 主管風格/願景/帶人 |
| 問題 | 他適不適合？ | 他要不要跟我走？ |
| 關鍵 | 硬知識傳遞 | 軟連結建立 |

**黑名單（一面不要搶二面的話）**：
- ❌ 主管個人特質
- ❌ 團隊長期願景
- ❌ 主管怎麼訓練新人
- ❌ 「我們主管很 nice」

## 💰 薪資速查表

### 三星制度

| 星 | 項目 | 說明 |
|----|------|------|
| ⭐ | 底薪 | 3 萬（職能固定；財金正式後擇優領） |
| ⭐⭐ | 績效抽成 | 含目標獎金（達自訂目標 +1,000） |
| ⭐⭐⭐ | 活動獎金 | 月度衝刺/季達標等 |

### 階級平均月薪

| 階級 | 平均月薪 | 責任條件 |
|------|---------|----------|
| 新人 | 4 - 5 萬 | 底薪 + 活動獎金 6-8k + 初期抽成 |
| 正式 | 5.5 - 8 萬 | 兩個月平均 40 萬 |
| 專員 | 6.5 - 12 萬 | — |
| 主任 | 8 - 15 萬 | — |
| 襄理 | 10 - 20 萬 | 兩個月平均 90 萬，月 ≥ 75 萬 |

### 初級儲備（第四星 Bonus）

```
🌱 主任才能加掛
・達 40 萬業績 +$5,000
・達 60 萬業績 +$10,000
・每 3 梯次挑 2 人
```

### 財金 vs 職能差異

```
職能：正式後底薪抽成都領
財金：正式後擇優領（底薪 vs 抽成哪個高領哪個）
     → 交叉點 35 萬
```

## 🧠 節奏感 · Lynn 的專業度建議

```
1. 固定名詞要熟練
   「墨凡股份有限公司」
   「電話銷售」「課程行銷顧問」
   → 卡詞還好，名詞卡會掉專業感

2. 每段中間留 1-2 秒停頓
   → 自己緩一下
   → 讓求職者消化上一段
```

**自我檢查三問**（聽自己電訪錄音時）：
1. 我有念錯或卡住的名詞嗎？
2. 我每段之間有停頓嗎？
3. 我有沒有越講越小聲？（緊張訊號）

## 🎯 成長路徑 · 肌肉記憶四階段

```
知道 → 做到 → 做好 → 精通
```

- **知道**：看過、聽過、記得這個概念
- **做到**：照著做，結果符合標準
- **做好**：做起來自然、不卡、有彈性
- **精通**：有自己的版本，能教別人

看完四集影片 = **知道**。  
打第一通電話 = 進入**做到**。  
別急，沒有人一開始就精通。

## 💪 心態口袋

### 被拒絕時

> 「求職者跟我們本來就有相互選擇的權利。」— Vincent

- 他不是拒絕你，是我們不合適
- 雙向選擇，你也在篩他
- 每一次拒絕都在校正你的判讀

### 卡詞時

- 沒事，這是肌肉記憶的一部分
- 對練 → 聽錄音 → 修正，迭代就好
- 越講越順是必然的

### 緊張聲音變小時（育卉提到的自我察覺）

- 先深呼吸 2 秒
- 回到公版話稿的下一句
- 結束後聽錄音確認是哪個段落容易變小

## 📋 面試前 90 秒 Checklist

接電話 / 進面試前快速掃一遍：

```
☐ 對方是職能還是財金職缺？（制度不一樣）
☐ 應聘條件我有打開嗎？（判讀依據）
☐ 三階段順序我記得嗎？（標準→延伸→反問）
☐ 致命題黃金三句我背了嗎？
☐ 反問句我有準備至少 3 個嗎？
☐ 我現在聲音狀態 OK 嗎？（深呼吸）
```
