## 第 2 段（1:00-2:30）核心原則 · 曖昧不清沒人喜歡

**[畫面]** 一對男女尷尬對峙的簡筆圖

**[切 Vincent 原聲 V4-03]**  
> *「就像兩個人在一起,曖昧不清的氛圍沒人會喜歡,要嘛在一起要嘛不要。面對求職者也一樣,我們一定要跟她講清楚,這就是一份業務工作。」*

**[阿凱]**  
「Vincent 這句是整集 EP4 的靈魂:**曖昧不清的關係沒人喜歡**。」

**[大字]**
```
「要嘛在一起,要嘛不要。」
```

**[阿凱]**  
「新人最常犯的錯是 —— 害怕求職者嚇跑,所以講得含糊:  
『這個...看你想領多少啦...』  
『壓力...有時候會有啦...』  
『氛圍...大家都還不錯啦...』」

**[圖卡]**
```
❌ 曖昧回答的三大後果

1. 求職者感覺你在推銷/隱瞞
2. 對方不信任你,對話變假
3. 錄取後落差 → 試用期就跑
```

**[阿凱]**  
「記住,**誠實比推銷更有說服力**。這是 EP4 最重要的心態調整。」

## 第 3 段（2:30-4:30）致命題 1：業績壓力 · 錯誤示範

**[畫面]** 電話通話場景

**[阿凱]**  
「我們看一個真實案例。這是新人對練時候發生的真實對話 —— 」

**[切 Vincent 原聲 V4-02]**  
> *[Vincent] 「剛剛求職者有說到『這會有業績壓力嗎?』。如果今天再重新來一次,妳會怎麼回覆?」  
> [受訓 HRBP] 「我剛剛好像沒有很正面回答她有沒有業績壓力。好像有點拐一個彎在回答。我不太知道怎麼明確跟她說有還是沒有,因為我的理解業務就是想要賺更多,就會更努力工作。」*

**[阿凱]**  
「聽出問題了嗎?這位新人沒有**正面回答**,而是用『取決於妳想領多少』來迴避。  
這看起來很聰明,其實是**典型曖昧**。求職者的感受是:『欸所以到底有沒有壓力?』」

**[圖卡:常見曖昧語言]**
```
⚠️ 新人最常誤用的迴避語

・「看妳想領多少」
・「每個人感受不同」
・「可能會有一點」
・「要看妳怎麼定義」
```

**[阿凱]**  
「這些話聽起來中性,但**把球踢回給求職者了**。求職者想問的是事實,不是哲學。」

## 第 4 段（4:30-6:30）致命題 1：業績壓力 · Vincent 正解

**[畫面]** 切到 Vincent 示範場景

**[切 Vincent 原聲 V4-04]**  
> *「我們可以直接跟她說:有,一定會有業績壓力。業務就是有業績壓力才叫業務。但我們不需要隱瞞或拐彎,我們要讓她知道:我們怎麼幫妳化解壓力。」*

**[阿凱]**  
「這就是示範。請把這三句話背起來:」

**[大圖卡]**
```
🎯 業績壓力黃金三句

1.「有,一定會有業績壓力」
   → 正面承認,不迴避

2.「業務就是有業績壓力才叫業務」
   → 用產業本質合理化

3.「重點是我們怎麼幫妳化解」
   → 轉向支持系統
```

**[阿凱]**  
「第一句話直接坦承,求職者會**瞬間信任你**,因為你沒有在賣假貨。  
第二句話讓壓力變『產業常態』,不是我們獨有。  
第三句話轉向正面,告訴對方你不是丟她自己扛。」

**[切 Vincent 原聲 V4-05]**  
> *「我們有完善的教育訓練。第一週的學習課表:第一天學什麼、第二天學什麼。第二週學長姐帶著妳開單。面對求職者,她想嘗試但又怕壓力巨大。我們可以直接說:有壓力,但我們有完善的教育訓練制度幫妳面對。這會給予安心感。」*

**[圖卡:安心感話術公式]**
```
有壓力(承認) 
   ↓
+ 第一週課表(具體)
   ↓
+ 第二週學長姊帶單(具體)
   ↓
= 不是丟妳自己扛(安心感)
```

**[阿凱]**  
「記住這條公式。當你把『第一週怎麼帶、第二週怎麼帶』講具體了,壓力就從**未知的大怪物**變成**可預見的挑戰**。」

## 第 5 段（6:30-8:00）致命題 2：團隊氛圍 · 先懂三件事

**[畫面]** 切到團隊氛圍場景

**[切 Vincent 原聲 V4-06]**  
> *「我們要先做到三件事:第一,針對團隊事實來陳述,否則進來發現事實不符會炸掉。第二,對事實如何做到陳述?在招募表單上都有團隊相關的文化,還有我貼的一面篩選標準。第三,按照上面的內容去講我們團隊需要的是哪一種人。」*

**[阿凱]**  
「團隊氛圍比業績壓力更難答,因為『氛圍』太抽象。Vincent 的答法是 —— **從抽象拉到具體,從主觀拉到事實**。」

**[大圖卡]**
```
💡 團隊氛圍三步驟

① 陳述事實(不是形容)
   ✗ 大家都很 nice
   ✓ 我們節奏快、互相幫忙

② 用應聘條件當依據
   → 二面主管訂的條件就是我們團隊長相

③ 講團隊需要什麼樣的人
   → 把『氛圍』轉成『我們要誰』
```

**[阿凱]**  
「第②點是**神來之筆**。應聘條件是主管訂的,上面寫的特質就是團隊的真實樣貌 —— 勇於試錯、勇於承擔、熱愛知識、熱於分享。這些不是形容詞,是**篩選標準**。」

## 第 6 段（8:00-9:30）致命題 2：團隊氛圍 · 完整示範

**[切 Vincent 原聲 V4-07]**  
> *「我們的團隊氣氛並不是說很壓迫,但是節奏很快,大家都在做事也會相互幫忙。業績壓力勢必會有,因為這是業務,但我們是有節奏地在推,不會是被罵的那種。你願意做、願意學,我們也會幫你一起把業績跟目標做出來,而不是丟給你自己去扛。」*

**[阿凱]**  
「這段拆解給你看 —— 」

**[圖卡:Vincent 示範拆解]**
```
句子 1:「並不是壓迫,但節奏快」
→ 定調(不是極端)+ 具體(節奏快)

句子 2:「大家都在做事也會互相幫忙」
→ 具象畫面,不是形容詞

句子 3:「壓力勢必會有,業務都這樣」
→ 承認事實

句子 4:「有節奏地推,不是被罵那種」
→ 緩解恐懼

句子 5:「你願意做、我們幫你一起扛」
→ 承諾支持
```

**[阿凱]**  
「整段不用任何『很好』『不錯』這種空話。每一句都是**事實 + 畫面**。這就是專業 HRBP 的答法。」
