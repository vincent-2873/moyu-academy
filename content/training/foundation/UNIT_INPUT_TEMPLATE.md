# 單元輸入模板（UNIT INPUT TEMPLATE）

> **使用說明：**
> 1. 複製本檔內容，存成 `units/{體系}/{單元代號}/input.md`
> 2. 填寫下方所有欄位（標 `[必填]` 的一定要有，`[選填]` 可空）
> 3. 填完後把整份丟給 Claude（對話）→ 產出腳本、網頁、手冊
> 4. 也可以直接丟給 Claude Code，它會依照 `TRAINING_MASTER.md` + `{體系}_MODULE_SPEC.md` 自動產出

---

## 📌 基本資訊

- **單元代號**：`[必填]` 例如 `HR-011`
- **單元名稱**：`[必填]` 例如「請假規則與操作」
- **所屬體系**：`[必填]` HR / BIZ / LEGAL
- **受眾代號**：`[必填]` ALL / NEW / MGR / HR-INT（可複選）
- **優先級**：`[必填]` P0 / P1 / P2
- **預估總時長**：`[必填]` 例如「影片 4 分鐘 + 互動 8 分鐘」
- **前置單元**：`[選填]` 例如「HR-001」（沒有就寫「無」）

## 🎨 產出需求

### 影片腳本（必產出）
- 主講角色：`[必填]` 阿凱 / 小美 / Ray / 情境劇（多人）
- 風格偏好：`[選填]` 例如「輕鬆一點、可加冷笑話」「嚴肅專業」
- 時長限制：`[選填]` 預設 3–5 分鐘

### 互動網頁（必產出）
- 題數：`[選填]` 預設 3 題，可要求更多
- 題型偏好：`[選填]` 情境判斷 / 對錯題 / 流程排序 / 分支選擇
- 特殊互動：`[選填]` 例如「最後要有一個情境分支選擇」

### 文件手冊（視情況）
- 是否產出：`[必填]` 是 / 否
- 特殊需求：`[選填]` 例如「要含附錄：假別對照表」
